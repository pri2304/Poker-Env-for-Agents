# Poker Env for Agents

A work-in-progress poker environment designed for experimenting with different types of decision-making agents.

The goal of this project is to provide a common environment where multiple agent architectures can be evaluated and compared under the same game conditions.

## Current Focus

The environment is being built to support:

* Rule-based agents
* Algorithmic agents
* Reinforcement Learning (RL) agents
* LLM-powered agents

## Project Goals

* Standardized poker game environment
* State representation for agent decision making
* Agent evaluation and benchmarking
* Support for multiple player strategies
* Future RL training integration

## Current Status

⚠️ This project is still under active development and is not feature-complete.

Several core systems are implemented, but the architecture and state representation are still being refined before larger-scale training experiments are conducted.
